# Test Scenarios - User Management API

> **Test Scenario = que validar**.

## SCN-001 - Crear usuario exitosamente
Validar que la API permita crear un usuario con datos validos y retorne identificador y timestamp.

## SCN-002 - Consultar usuario existente
Validar que la API retorne los datos de un usuario existente con contrato correcto.

## SCN-003 - Consultar usuario inexistente
Validar que la API responda correctamente ante usuario no existente.

## SCN-004 - Actualizar usuario exitosamente
Validar que la API actualice los datos de un usuario y retorne los campos esperados.

## SCN-005 - Eliminar usuario exitosamente
Validar que la API elimine un usuario y responda con codigo sin contenido.

## SCN-006 - Cumplimiento de headers y tiempos
Validar headers de contenido y tiempos de respuesta bajo umbral definido.
